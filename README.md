#OIBSIP_WebDevelopmentAndDesigning_Task2
## Tribute Page – Savitribai Phule
This project is a Tribute Page dedicated to Savitribai Phule, a great social reformer and pioneer of women's education in India.
It is created using HTML and CSS as part of the Oasis Infobyte Web Development and Designing Internship (OIBSIP).
 Objective
The objective of this task is to design a simple and visually appealing tribute page for a person I admire using basic HTML and CSS concepts.

## Technologies Used
HTML
CSS
Google Fonts
Visual Studio Code
Web Browser (Google Chrome)

## Project Files
index.html – Structure and content of the tribute page
style.css – Styling and layout design of the webpage

## Features
Clean and responsive layout
Image and information about Savitribai Phule
Inspirational quote section
Achievement cards highlighting major contributions
Modern UI with Google Fonts and CSS styling

## Outcome
Successfully created a Tribute Page website that highlights the life, achievements, and contributions of Savitribai Phule.
This project helped in improving my understanding of HTML structure, CSS styling, and webpage layout design.
